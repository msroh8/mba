# 💡 المولد الذكي للمبادرات المدرسية

تطبيق يساعد المعلمين على صياغة مبادرات مدرسية احترافية باستخدام الذكاء الاصطناعي.

## 🚀 تشغيل محلي

```bash
pip install -r requirements.txt
streamlit run mba.py
```

## ☁️ نشر على Streamlit Cloud

1. ارفع المشروع على GitHub
2. افتح [share.streamlit.io](https://share.streamlit.io) وربط المستودع
3. أضف المفتاح في **Settings > Secrets**:

```toml
GEMINI_API_KEY = "your-key-here"
```

> ⚠️ لا ترفع مفتاح API مباشرة في الكود أبداً
